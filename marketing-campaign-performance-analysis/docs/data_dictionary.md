# Data Dictionary

## Dataset Overview

This project analyzes marketing campaign performance across three beauty e-commerce brands:
- Nykaa
- Purplle
- Tira Beauty

The dataset contains campaign-level performance metrics including engagement, conversions, revenue, and customer segmentation information.


## Columns Description

| Column | Description | Data Type |
|---|---|---|
| Campaign_ID | Unique identifier for each marketing campaign | Text |
| Campaign_Type | Type of marketing campaign | Text |
| Target_Audience | Audience targeted by the campaign | Text |
| Duration | Campaign duration in days | Integer |
| Channel_Used | Marketing channels used | Text |
| Impressions | Number of times the campaign was displayed | Integer |
| Clicks | Number of user clicks generated | Integer |
| Leads | Number of potential customers captured | Integer |
| Conversions | Number of successful conversions | Integer |
| Revenue | Revenue generated from the campaign | Numeric |
| Acquisition_Cost | Cost associated with acquiring customers | Numeric |
| ROI | Return on investment metric | Numeric |
| Language | Language used in the campaign | Text |
| Engagement_Score | Engagement level score | Numeric |
| Customer_Segment | Customer segment targeted | Text |
| Date | Campaign date | Date |
