# Marketing Campaign Performance Analysis

## Project Overview

This project analyzes marketing campaign performance data to identify key drivers of revenue, conversions, and return on investment (ROI).

The goal is to transform raw campaign data into actionable business insights through data cleaning, SQL analysis, and interactive Power BI dashboards.

---

## Business Questions

The analysis focuses on answering:

- Which brands generate the highest revenue?
- Which marketing channels deliver the best ROI?
- Which campaign types perform better?
- Which customer segments contribute the most revenue?
- Which campaigns and channel combinations are the most effective?
- How does performance change over time?

## Data Pipeline

The project workflow:

1. Raw campaign datasets were collected and combined.
2. Data quality checks were performed to identify missing values and inconsistencies.
3. Data was cleaned and prepared using Python.
4. SQL queries were developed to analyze campaign performance.
5. Business insights were extracted from campaign, channel, brand, and customer segment data.
6. Interactive dashboards were created in Power BI.

## Key Insights

Examples of insights identified:

- Revenue contribution is relatively balanced across the analyzed brands.
- Email, YouTube, and WhatsApp showed strong ROI performance compared with other channels.
- Campaign effectiveness varies depending on channel combinations.
- Customer segments contribute differently to overall revenue performance.
- Monitoring ROI and CPA together provides a clearer view of campaign efficiency.

---

## Recommendations

Based on the analysis:

- Prioritize high-performing channels and campaign combinations.
- Continuously monitor ROI together with acquisition cost.
- Optimize budget allocation toward campaigns with stronger conversion efficiency.
- Use customer segment insights to improve targeting strategies.
